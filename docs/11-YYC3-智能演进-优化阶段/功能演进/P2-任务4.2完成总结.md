## 🔄 使用示例

### 性能监控
```typescript
import { PerformanceMonitor } from './llm/PerformanceMonitor';

const monitor = new PerformanceMonitor({
  interval: 1000,
  thresholds: {
    renderTime: 16,
    memoryUsage: 80,
  },
});

// 启动监控
monitor.start();

// 监控渲染
monitor.monitorRender('MyComponent', 12);

// 获取统计
const stats = monitor.getStats();
console.log('平均渲染时间:', stats.averageRenderTime);
```
