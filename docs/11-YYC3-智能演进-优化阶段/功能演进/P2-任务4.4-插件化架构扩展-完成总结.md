- 可配置的全局变量白名单

- 网络请求白名单
- 超时控制
- 资源监控（内存、CPU）
- 沙箱清理

#### 3.3 PluginLoader (插件加载器)
- **加载流程**:
  1. 验证清单（ID、名称、版本、作者等）
  2. 验证签名（可选）
  3. 检查依赖（缺失、版本不匹配）
  4. 创建初始状态
  5. 更新状态为 LOADING
  6. 加载插件代码
  7. 更新状态为 LOADED

- **激活流程**:
  1. 更新状态为 ACTIVATING
  2. 创建插件上下文
  3. 执行激活钩子
  4. 注册扩展点贡献
  5. 更新状态为 ACTIVE

- **停用流程**:
  1. 更新状态为 DEACTIVATING
  2. 执行停用钩子
  3. 清理订阅
  4. 清理扩展点贡献
  5. 更新状态为 DISABLED

**特点**:
- ✅ 完整的加载、激活、停用流程

## 🚀 使用示例

### 1. 创建插件清单

```typescript
const myPlugin: PluginManifest = {
  id: 'my-plugin',
  name: 'My Plugin',
  version: '1.0.0',
  description: 'A sample plugin',
  author: 'Developer',
  permissions: [
    PluginPermission.FILE_READ,
    PluginPermission.EDITOR_WRITE,
    PluginPermission.UI_PANEL,
  ],
  dependencies: [
    { id: 'core-plugin', version: '^1.0.0' },
  ],
  activationEvents: [
    { type: 'onCommand', command: 'myPlugin.activate' },
  ],
  contributes: {
    commands: [
      { command: 'myPlugin.hello', title: 'Say Hello' },
    ],
    views: [
      { id: 'myView', name: 'My View' },
    ],
  },
  activate: (context) => {
    context.commands.registerCommand('myPlugin.hello', () => {
      context.ui.showNotification('Hello from plugin!', 'info');
    });
  },
  deactivate: () => {
    console.log('Plugin deactivated');
  },
};
```

### 2. 加载和激活插件

```typescript
// 初始化组件
const lifecycleManager = new LifecycleManager();
const dependencyManager = new DependencyManager();
const extensionPointManager = new ExtensionPointManager();
initializeDefaultExtensionPoints(extensionPointManager);

const loader = new PluginLoader({
  lifecycleManager,
  dependencyManager,
  extensionPointManager,
  verifySignatures: true,
});

const apiFactory = new PluginAPIFactory();

// 加载插件
const loadResult = await loader.loadPlugin(myPlugin, {
  installedPlugins: new Map(),
});

if (loadResult.success) {
  // 激活插件
  const activateResult = await loader.activatePlugin(
    loadResult.plugin!,
    () => apiFactory.createAPI('my-plugin'),
  );
  
  if (activateResult.success) {
    console.log('Plugin activated!');
  }
}
```
