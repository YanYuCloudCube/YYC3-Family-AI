## 🎯 核心特性

### 1. 智能意图识别

```
用户输入 → 意图检测 → 意图特定指导 → 系统提示词
```

**支持8种意图**:
- **generate**: 生成新代码/组件
- **modify**: 修改现有代码
- **fix**: 修复错误/bug
- **explain**: 解释代码
- **refactor**: 重构优化
- **test**: 生成测试
- **review**: 代码审查
- **general**: 通用对话

### 2. 上下文感知

```
项目上下文 → Token 压缩 → 智能注入 → 系统提示词
```

**上下文组成**:
- 文件树结构
- 活跃文件内容
- Git 状态信息
- 打开的文件列表
- 相关文件内容

### 3. 多层次注入

```
基础角色 + 意图指导 + 输出格式 + 项目上下文 + 自定义指令
```

**集成点**:
- 智能体角色提示词（来自 Settings Store）
- 编码规则与约束（来自规则管理）
- 可用技能描述（来自技能管理）

```typescript
const INTENT_PATTERNS = {

    /生成.*测试|写.*测试|单元测试/,
    /generate.*test|write.*test|unit test/i,
  ],
  // ...
};
```

### 2. 智能上下文压缩

```typescript
// Token 估算算法
const estimateTokens = (text: string) => {
  const chineseChars = (text.match(/[\u4e00-\u9fa5]/g) || []).length;
  const otherChars = text.length - chineseChars;
  return Math.ceil(chineseChars / 2 + otherChars / 4);
};
```
