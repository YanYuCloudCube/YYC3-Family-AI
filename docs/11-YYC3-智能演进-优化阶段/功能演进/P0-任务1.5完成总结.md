## 🔧 技术亮点

### 1. 联合类型优于枚举

```typescript
// ✅ 推荐 - 简洁、类型安全
type PreviewMode = "realtime" | "manual" | "delayed";

// ❌ 避免 - 冗余、需要转换
enum PreviewMode {
  Realtime = "realtime",
  Manual = "manual",
  Delayed = "delayed",
}
```

### 2. 可选参数标注

```typescript
interface SnapshotManagerConfig {
  maxSnapshots?: number;  // 可选参数
  storageKey?: string;    // 可选参数
}

// 使用时提供默认值
const config: SnapshotManagerConfig = {
  maxSnapshots: options.maxSnapshots ?? 50,
};
```

### 3. 类型复用

```typescript
// 基础类型
type PreviewMode = "realtime" | "manual" | "delayed";

// 复用基础类型
interface PreviewModeConfig {
  mode: PreviewMode;  // 复用
  delay?: number;
}
```

### 4. 文档与代码同步

```typescript
/**
 * 预览模式类型
 * 
 * - realtime: 实时模式 - 代码变更立即触发预览更新
 * - manual: 手动模式 - 用户手动触发预览更新
 * - delayed: 延迟模式 - 代码变更后延迟一定时间再触发预览更新
 */
type PreviewMode = "realtime" | "manual" | "delayed";
```
