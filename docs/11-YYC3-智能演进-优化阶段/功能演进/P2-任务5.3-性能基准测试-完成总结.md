```typescript
interface PerformanceReport {
  id: string;                           // 报告ID
  timestamp: number;                    // 生成时间
  version: string;                      // 版本号

  comparison?: PerformanceComparisonResult; // 性能对比（可选）
  issues: PerformanceIssue[];           // 问题列表
  recommendations: PerformanceRecommendation[]; // 建议
  summary: string;                      // 总结
}
```

### 5. 性能评分系统

**评分标准**:
- **Excellent** (90-100分): 性能卓越
- **Good** (70-89分): 性能良好
- **Warning** (50-69分): 性能警告
- **Critical** (0-49分): 性能严重问题

**评分计算**:
```typescript
const scores = testResults.map(result => {
  switch (result.level) {
    case 'excellent': return 100;
    case 'good': return 80;
    case 'warning': return 60;
    case 'critical': return 40;
    default: return 0;
  }
});

return scores.reduce((a, b) => a + b, 0) / scores.length;
```
