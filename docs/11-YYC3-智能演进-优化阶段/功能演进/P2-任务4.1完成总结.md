## 📝 使用示例

```typescript
import { ContextCompressor } from './llm/ContextCompressor';

// 创建压缩器
const compressor = new ContextCompressor({
  strategy: CompressionStrategyType.INTELLIGENT,
  maxTokenLimit: 4000,
  preserveStructure: true,
});

// 压缩代码
const result = compressor.compress(largeCodeFile);

// 查看结果
console.log('压缩率:', result.compressionRatio + '%');
```
