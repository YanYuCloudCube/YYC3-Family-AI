### 1.3.3 主题同步策略 ✅

**文件**: `ThemeSyncManager.ts` (360行)

**核心功能**:
- ✅ "跟随系统"选项（自动同步系统主题）
- ✅ 手动模式（用户手动选择主题）
- ✅ 手动覆盖（自动模式下临时覆盖）
- ✅ 用户偏好持久化和恢复
- ✅ 首次访问智能初始化

**同步策略**:

#### 自动模式（跟随系统）
```
系统主题 → 应用主题映射：
  dark  → navy (深海军蓝)
  light → light (浅色主题)
```

#### 手动模式
```
用户选择的主题立即生效，不受系统主题影响
```

#### 手动覆盖（自动模式下）
```
允许用户在自动模式下临时覆盖主题
可随时清除覆盖，恢复自动同步
```

**配置选项**:
```typescript
interface ThemeSyncConfig {
  enableAutoSync: boolean;              // 是否启用自动同步
  followSystemOnFirstVisit: boolean;    // 首次访问是否跟随系统
  fallbackTheme: ThemeType;             // 不支持系统主题时的默认主题
  showManualOverrideHint: boolean;      // 是否显示手动覆盖提示
}
```

**API接口**:
```typescript
initialize(): void                      // 初始化同步管理器
setTheme(theme): void                   // 设置主题
setSyncMode(mode): void                 // 设置同步模式
toggleSyncMode(): ThemeSyncMode         // 切换同步模式
clearManualOverride(): void             // 清除手动覆盖
getSyncState(): ThemeSyncState          // 获取同步状态
reset(): void                           // 重置设置
```
