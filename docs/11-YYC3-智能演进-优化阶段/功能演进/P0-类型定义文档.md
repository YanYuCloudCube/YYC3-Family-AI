## PreviewModeController 类型

### PreviewMode

预览模式类型定义。

```typescript
type PreviewMode = "realtime" | "manual" | "delayed";
```

**说明**:
- `realtime`: 实时模式 - 代码变更立即触发预览更新
- `manual`: 手动模式 - 用户手动触发预览更新
- `delayed`: 延迟模式 - 代码变更后延迟一定时间再触发预览更新

**示例**:
```typescript
const mode: PreviewMode = "realtime";

// 模式切换
function switchMode(newMode: PreviewMode) {
  console.log(`切换到 ${newMode} 模式`);
}
```

### PreviewModeConfig

预览模式配置。

```typescript
interface PreviewModeConfig {
  /** 当前预览模式 */
  mode: PreviewMode;
  /** 延迟时间（毫秒），仅 delayed 模式有效 */
  delay?: number;
  /** 是否自动保存 */
  autoSave?: boolean;
}
```

**示例**:
```typescript
const config: PreviewModeConfig = {
  mode: "delayed",
  delay: 1000,
  autoSave: true,
};
```

### PreviewModeControllerConfig

预览模式控制器配置。

```typescript
interface PreviewModeControllerConfig {
  /** 初始模式 */
  initialMode?: PreviewMode;
  /** 延迟模式的延迟时间（毫秒） */
  delayMs?: number;
  /** 更新回调函数 */
  onUpdate?: () => void;
}
```

**示例**:
```typescript
const controllerConfig: PreviewModeControllerConfig = {
  initialMode: "realtime",
  delayMs: 1000,
  onUpdate: () => {
    console.log("预览已更新");
  },
};

const controller = new PreviewModeController(controllerConfig);
```

## SnapshotManager 类型

### SnapshotFile

快照文件定义。

```typescript
interface SnapshotFile {
  /** 文件路径 */
  path: string;
  /** 文件内容 */
  content: string;
  /** 内容哈希值（用于快速比较） */
  hash?: string;
}
```

**示例**:
```typescript
const file: SnapshotFile = {
  path: "src/App.tsx",
  content: "export default function App() {}",
  hash: "abc123",
};
```

### SnapshotMetadata

快照元数据。

```typescript
interface SnapshotMetadata {
  /** 快照ID */
  id: string;
  /** 快照标签 */
  label: string;
  /** 创建时间戳 */
  createdAt: number;
  /** 文件数量 */
  fileCount: number;
  /** 总字符数 */
  totalChars: number;
  /** 总行数 */
  totalLines: number;
  /** 标签列表 */
  tags?: string[];
  /** 描述 */
  description?: string;
}
```

**示例**:
```typescript
const metadata: SnapshotMetadata = {
  id: "snap-001",
  label: "功能A完成",
  createdAt: Date.now(),
  fileCount: 5,
  totalChars: 1200,
  totalLines: 48,
  tags: ["feature", "stable"],
  description: "完成了用户认证功能",
};
```

### Snapshot

完整快照定义。

```typescript
interface Snapshot {
  /** 快照元数据 */
  metadata: SnapshotMetadata;
  /** 快照文件列表 */
  files: SnapshotFile[];
}
```

**示例**:
```typescript
const snapshot: Snapshot = {
  metadata: {
    id: "snap-001",
    label: "初始版本",
    createdAt: Date.now(),
    fileCount: 2,
    totalChars: 500,
    totalLines: 20,
  },
  files: [
    {
      path: "src/App.tsx",
      content: "export default function App() {}",
    },
    {
      path: "src/index.tsx",
      content: "import App from './App';",
    },
  ],
};
```

### SnapshotDiff

快照比较结果。

```typescript
interface SnapshotDiff {
  /** 新增的文件 */
  added: SnapshotFile[];
  /** 删除的文件路径 */
  removed: string[];
  /** 修改的文件 */
  modified: Array<{
    path: string;
    oldContent: string;
    newContent: string;
    oldHash: string;
    newHash: string;
  }>;
  /** 未变化的文件路径 */
  unchanged: string[];
}
```

**示例**:
```typescript
const diff: SnapshotDiff = {
  added: [
    { path: "src/utils.ts", content: "export function helper() {}" },
  ],
  removed: ["src/old-file.ts"],
  modified: [
    {
      path: "src/App.tsx",
      oldContent: "export default function App() {}",
      newContent: "export default function App() { return <div /> }",
      oldHash: "abc123",
      newHash: "def456",
    },
  ],
  unchanged: ["src/index.tsx"],
};
```

## CodeValidator 类型

### ValidationResult

代码验证结果。

```typescript
interface ValidationResult {
  /** 是否通过验证 */
  valid: boolean;
  /** 警告信息列表 */
  warnings: string[];
  /** 错误信息列表 */
  errors: string[];
  /** 建议信息列表 */
  suggestions: string[];
  /** 代码指标 */
  metrics: {
    /** 行数 */
    lines: number;
    /** 字符数 */
    characters: number;
    /** 复杂度：低/中/高 */
    complexity: "low" | "medium" | "high";
  };
}
```

**示例**:
```typescript
const result: ValidationResult = {
  valid: true,
  warnings: ["存在多个 console.log (6 个)，建议清理"],
  errors: [],
  suggestions: ["使用了 any 类型，建议使用具体类型"],
  metrics: {
    lines: 48,
    characters: 1200,
    complexity: "medium",
  },
};
```

### ParsedCodeBlock

解析后的代码块。

```typescript
interface ParsedCodeBlock {
  /** 文件路径 */
  filepath: string;
  /** 代码内容 */
  content: string;
  /** 语言类型 */
  language: string;
  /** 是否为新文件 */
  isNewFile: boolean;
}
```

**示例**:
```typescript
const block: ParsedCodeBlock = {
  filepath: "src/components/Button.tsx",
  content: "export function Button() { return <button>Click</button> }",
  language: "tsx",
  isNewFile: true,
};

// 验证代码块
const validator = new CodeValidator();
const result = validator.validate(block);
```

## SystemPromptBuilder 类型

### UserIntent

用户意图类型。

```typescript
type UserIntent =
  | "generate"   // 生成新代码/组件
  | "modify"     // 修改现有代码
  | "fix"        // 修复错误/bug
  | "explain"    // 解释代码
  | "refactor"   // 重构优化
  | "test"       // 生成测试
  | "review"     // 代码审查
  | "general";   // 通用对话
```

**示例**:
```typescript
import { detectIntent } from '@/app/components/ide/ai/SystemPromptBuilder';

const userMessage = "创建一个登录表单组件";
const intent: UserIntent = detectIntent(userMessage);
// intent = "generate"
```

### SystemPromptConfig

系统提示词配置。

```typescript
interface SystemPromptConfig {
  /** 用户意图 */
  intent: UserIntent;
  /** 项目上下文（可选） */
  context?: ProjectContext | null;
  /** 自定义指令（可选） */
  customInstructions?: string;
  /** 最大 Token 数（可选） */
  maxTokens?: number;
}
```

**示例**:
```typescript
const config: SystemPromptConfig = {
  intent: "generate",
  context: projectContext,
  customInstructions: "请使用 Tailwind CSS",
  maxTokens: 8000,
};

const prompt = buildSystemPrompt(config.intent, config.context, {
  customInstructions: config.customInstructions,
  maxContextTokens: config.maxTokens,
});
```

## 使用示例

### 示例1: 完整的预览模式控制流程

```typescript
import { PreviewModeController, PreviewMode } from '@/app/components/ide/PreviewModeController';

// 创建控制器
const controller = new PreviewModeController({
  initialMode: "realtime",
  delayMs: 1000,
  onUpdate: () => {
    console.log("预览已更新");
  },
});

// 设置模式
controller.setMode("delayed");

// 触发更新
controller.triggerUpdate();

// 获取当前模式
const currentMode: PreviewMode = controller.getMode();
```

### 示例2: 快照管理完整流程

```typescript
import { SnapshotManager, Snapshot, SnapshotDiff } from '@/app/components/ide/SnapshotManager';

// 创建管理器
const manager = new SnapshotManager({
  maxSnapshots: 50,
  storageKey: "my-snapshots",
});

// 创建快照
const snapshot = manager.createSnapshot("功能A完成", [
  { path: "src/App.tsx", content: "..." },
  { path: "src/utils.ts", content: "..." },
]);

// 列出所有快照
const snapshots: Snapshot[] = manager.listSnapshots();

// 比较快照
const diff: SnapshotDiff = manager.compareSnapshots("snap-001", "snap-002");

// 恢复快照
manager.restoreSnapshot("snap-001");

// 删除快照
manager.deleteSnapshot("snap-002");
```

### 示例3: 代码验证完整流程

```typescript
import { CodeValidator, ValidationResult, ParsedCodeBlock } from '@/app/components/ide/CodeValidator';

// 创建验证器
const validator = new CodeValidator();

// 准备代码块
const block: ParsedCodeBlock = {
  filepath: "src/components/Button.tsx",
  content: `
    export function Button() {
      return <button className="btn">Click</button>;
    }
  `,
  language: "tsx",
  isNewFile: true,
};

// 验证代码
const result: ValidationResult = validator.validate(block);

// 检查结果
if (!result.valid) {
  console.error("验证失败:", result.errors);
}

if (result.warnings.length > 0) {
  console.warn("警告:", result.warnings);
}

if (result.suggestions.length > 0) {
  console.info("建议:", result.suggestions);
}

console.log("指标:", result.metrics);
```

## 最佳实践

### 1. 类型安全

始终使用 TypeScript 类型注解：

```typescript
// ✅ 推荐
const mode: PreviewMode = "realtime";

// ❌ 避免
const mode = "realtime"; // 类型推断为 string，不安全
```

### 2. 配置对象

使用配置对象而非多个参数：

```typescript
// ✅ 推荐
const controller = new PreviewModeController({
  initialMode: "realtime",
  delayMs: 1000,
});

// ❌ 避免
const controller = new PreviewModeController("realtime", 1000);
```

### 3. 可选参数

明确标注可选参数：

```typescript
// ✅ 推荐
interface SnapshotManagerConfig {
  maxSnapshots?: number; // 可选参数
  storageKey?: string;   // 可选参数
}

// 使用时提供默认值
const config: SnapshotManagerConfig = {
  maxSnapshots: options.maxSnapshots ?? 50,
};
```

### 4. 联合类型

使用联合类型而非枚举：

```typescript
// ✅ 推荐
type PreviewMode = "realtime" | "manual" | "delayed";

// ❌ 避免
enum PreviewMode {
  Realtime = "realtime",
  Manual = "manual",
  Delayed = "delayed",
}
```

### 5. 类型守卫

使用类型守卫进行类型检查：

```typescript
function isValidPreviewMode(value: string): value is PreviewMode {
  return ["realtime", "manual", "delayed"].includes(value);
}

const userInput = "realtime";
if (isValidPreviewMode(userInput)) {
  // TypeScript 知道 userInput 是 PreviewMode 类型
  controller.setMode(userInput);
}
```
