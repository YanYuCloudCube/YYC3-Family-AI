**文件大小**: 430行  
**核心功能**:

#### 同步模式
```typescript
enum SyncMode {
  DISABLED = 'disabled',              // 禁用同步
  EDITOR_TO_PREVIEW = 'editor_to_preview',  // 编辑器 → 预览
  PREVIEW_TO_EDITOR = 'preview_to_editor',  // 预览 → 编辑器
  BIDIRECTIONAL = 'bidirectional',    // 双向同步
}
```

#### 核心特性
- ✅ **编辑器滚动同步到预览** - 实时同步编辑器滚动位置
- ✅ **预览滚动同步到编辑器** - 实时同步预览滚动位置
- ✅ **单向/双向同步** - 灵活配置同步方向
- ✅ **同步阈值设置** - 避免频繁同步，提升性能

#### 高级功能
- **防抖机制**: 可配置的防抖延迟（默认50ms）
- **平滑滚动**: 支持平滑滚动效果
- **按比例同步**: 支持按比例同步不同尺寸的元素
- **防循环保护**: 300ms防循环时间窗口
- **事件监听**: 完整的事件通知系统

#### 配置选项
```typescript
interface ScrollSyncConfig {
  mode: SyncMode;            // 同步模式
  threshold: number;         // 同步阈值（像素）
  debounceDelay: number;     // 防抖延迟（毫秒）
  smoothScroll: boolean;     // 是否平滑滚动
  syncRatio: boolean;        // 是否按比例同步
}
```

### 3. 视图状态持久化（ViewStatePersistence.ts）

**文件大小**: 410行  
**核心功能**:

#### 状态保存
- ✅ **保存滚动位置** - 记录横向和纵向滚动位置
- ✅ **保存缩放级别** - 记录当前缩放级别
- ✅ **自动过期清理** - 基于TTL自动清理过期状态
- ✅ **数量限制** - 最多保存100个状态

#### 视图状态接口
```typescript
interface ViewState {
  id: string;                      // 状态ID
  filePath: string;                // 文件路径
  scrollPosition: ScrollPosition;  // 滚动位置
  zoomLevel: number;               // 缩放级别
  timestamp: number;               // 时间戳
  ttl?: number;                    // 存活时间（毫秒）
}
```

#### 核心方法
```typescript
// 保存和恢复
saveState(filePath, scrollPosition, zoomLevel, ttl?): ViewState
restoreState(filePath): ViewState | null

// 查找
findStateByFile(filePath): ViewState | null
getState(id): ViewState | null

// TTL管理
updateStateTTL(id, ttl): boolean
extendStateLife(id, additionalTTL): boolean

// 导入导出
exportStates(): string
importStates(json, merge?): number
```

#### 持久化配置
```typescript
interface PersistenceConfig {
  maxStates: number;        // 最大保存状态数（默认100）
  defaultTTL: number;       // 默认存活时间（7天）
  storageKey: string;       // localStorage键名
}
```

#### 智能特性
- **自动去重**: 同一文件路径只保留最新状态
- **过期清理**: 定期清理过期状态
- **数量限制**: 自动删除最旧的状态
- **错误恢复**: 损坏数据自动清理
- **存储优化**: 配额超限自动清理

## 📝 使用示例

### 滚动同步使用
```typescript
import { getScrollSyncEngine, SyncMode } from './preview/ScrollSyncEngine';

// 获取实例
const scrollEngine = getScrollSyncEngine({
  mode: SyncMode.BIDIRECTIONAL,
  threshold: 10,
  debounceDelay: 50,
});

// 设置元素
scrollEngine.setEditorElement(editorElement);
scrollEngine.setPreviewElement(previewElement);

// 添加监听器
scrollEngine.addListener((event) => {
  console.log('滚动事件:', event.source, event.position);
});

// 手动同步
scrollEngine.syncEditorToPreview();

// 更新配置
scrollEngine.updateConfig({ mode: SyncMode.EDITOR_TO_PREVIEW });
```

### 缩放控制使用
```typescript
import { getZoomController, setupZoomKeyboardShortcuts } from './preview/ZoomController';

// 获取实例
const zoomController = getZoomController({
  minZoom: 0.5,
  maxZoom: 2.0,
  zoomStep: 0.1,
});

// 设置目标元素
zoomController.setTargetElement(previewElement);

// 基本操作
zoomController.zoomIn();
zoomController.zoomOut();
zoomController.resetZoom();

// 设置预设缩放
zoomController.setPresetZoom('150%');

// 自适应缩放
zoomController.fitToContainer(800, 600);

// 添加监听器
zoomController.addListener((zoom) => {
  console.log('缩放级别:', zoom);
});

// 设置快捷键
const cleanup = setupZoomKeyboardShortcuts(zoomController);
```

### 视图状态持久化使用
```typescript
import { getViewStatePersistence } from './preview/ViewStatePersistence';

// 获取实例
const persistence = getViewStatePersistence({
  maxStates: 100,
  defaultTTL: 7 * 24 * 60 * 60 * 1000, // 7天
});

// 保存状态
persistence.saveState(
  '/src/app.ts',
  { x: 100, y: 200, ratioX: 0.5, ratioY: 0.5 },
  1.5
);

// 恢复状态
const state = persistence.restoreState('/src/app.ts');
if (state) {
  console.log('恢复滚动位置:', state.scrollPosition);
  console.log('恢复缩放级别:', state.zoomLevel);
}

// 导出所有状态
const exported = persistence.exportStates();

// 导入状态
persistence.importStates(exported, true);

// 延长状态生命周期
persistence.extendStateLife(state.id, 24 * 60 * 60 * 1000); // 延长1天
```
