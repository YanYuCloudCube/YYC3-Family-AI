## 🔄 使用示例

### WCAG合规检查
```typescript
import { WCAGChecker } from './llm/WCAGChecker';

const checker = new WCAGChecker({
  wcagLevel: WCAGLevel.AA,
});

// 检查整个页面
const report = checker.checkAll();

// 检查特定元素
const elementReport = checker.checkAll(containerElement);

// 查看报告
console.log('合规率:', report.summary.compliance + '%');
console.log('失败项:', report.summary.failed);
```

### 键盘导航
```typescript
import { KeyboardNavigator } from './llm/KeyboardNavigator';

const navigator = new KeyboardNavigator();

// 初始化导航
navigator.init(containerElement, {
  trapMode: FocusTrapMode.STRICT,
  autoFocus: true,
});

// 注册快捷键
navigator.registerNavigation('save', {
  type: NavigationType.SHORTCUT,
  keys: ['Control', 's'],
  handler: () => save(),
});
```

### ARIA管理
```typescript
import { ARIAManager } from './llm/ARIAManager';

const ariaManager = new ARIAManager();

// 设置ARIA属性
ariaManager.setAttributes(button, {
  role: ARIARole.BUTTON,
  label: 'Submit form',
  pressed: false,
});

// 通知屏幕阅读器
ariaManager.announce('Form submitted successfully', 'high');
```
