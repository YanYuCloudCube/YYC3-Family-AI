## 快速开始

### 安装依赖

P3 验证增强模块已经集成在项目主包中，无需额外安装。

### 基本使用

```typescript
import { ColorValidator } from './theme/ColorValidator';
import { FontSizeValidator } from './theme/FontSizeValidator';
import { ZoomController } from './preview/ZoomController';
import { SnapshotViewController } from './snapshot/SnapshotViewController';
import { BoundaryExceptionHandler } from './exception/BoundaryExceptionHandler';

// 创建验证器
const colorValidator = new ColorValidator();
const fontSizeValidator = new FontSizeValidator();
const zoomController = new ZoomController();

// 使用验证器
const colorResult = colorValidator.validateHex('#ff0000');
if (colorResult.valid) {
  console.log('颜色有效');
}

// 清理资源
colorValidator.destroy();
fontSizeValidator.destroy();
zoomController.destroy();
```

## ColorValidator 使用

### 验证十六进制颜色

```typescript
const validator = new ColorValidator();

// 验证 3 位格式
const result1 = validator.validateHex('#f00');
console.log(result1.valid); // true

// 验证 6 位格式
const result2 = validator.validateHex('#ff0000');
console.log(result2.valid); // true

// 验证无效格式
const result3 = validator.validateHex('ff0000');
console.log(result3.valid); // false
console.log(result3.error); // "Invalid hex color format"

validator.destroy();
```

### 验证 RGB 颜色

```typescript
const validator = new ColorValidator();

// 验证 RGB
const result1 = validator.validateRGB('rgb(255, 0, 0)');
console.log(result1.valid); // true

// 验证 RGBA
const result2 = validator.validateRGBA('rgba(255, 0, 0, 0.5)');
console.log(result2.valid); // true

// 验证超出范围的值
const result3 = validator.validateRGB('rgb(300, 0, 0)');
console.log(result3.valid); // false
console.log(result3.error); // "RGB value 300 is out of range (0-255)"
```

### 验证 HSL 颜色

```typescript
const validator = new ColorValidator();

// 验证 HSL
const result1 = validator.validateHSL('hsl(120, 100%, 50%)');
console.log(result1.valid); // true

// 验证 HSLA
const result2 = validator.validateHSLA('hsla(120, 100%, 50%, 0.5)');
console.log(result2.valid); // true

// 验证无效色相
const result3 = validator.validateHSL('hsl(400, 100%, 50%)');
console.log(result3.valid); // false
console.log(result3.error); // "HSL hue value 400 is out of range (0-360)"
```

### 验证颜色对比度（可访问性）

```typescript
const validator = new ColorValidator();

// 验证高对比度组合
const result1 = validator.validateContrast('#ffffff', '#000000');
console.log(result1.valid); // true
console.log(result1.contrastRatio); // 21.0
console.log(result1.wcagLevel); // 'AAA'

// 验证低对比度组合
const result2 = validator.validateContrast('#ffffff', '#eeeeee');
console.log(result2.valid); // false
console.log(result2.error); // "Contrast ratio 1.14 does not meet WCAG AA standard (4.5:1)"
console.log(result2.wcagLevel); // 'FAIL'

// 验证中等对比度组合
const result3 = validator.validateContrast('#0000ff', '#ffffff');
console.log(result3.valid); // true
console.log(result3.contrastRatio); // 8.59
console.log(result3.wcagLevel); // 'AAA'
```

### 批量验证颜色

```typescript
const validator = new ColorValidator();

// 定义主题颜色
const themeColors = {
  primary: '#ff0000',
  secondary: '#00ff00',
  background: '#000000',
  text: '#ffffff',
  accent: '#ffff00',
};

// 批量验证
const results = validator.validateAll(themeColors);

// 检查所有颜色是否有效
const allValid = results.every(r => r.valid);
console.log('所有颜色有效:', allValid);

// 显示无效颜色
results.forEach((result, index) => {
  if (!result.valid) {
    console.log(`无效颜色 ${index}:`, result.error);
  }
});
```

### 主题自定义流程

```typescript
import { useThemeStore } from './stores/useThemeStore';
import { ColorValidator } from './theme/ColorValidator';

// 1. 创建验证器
const colorValidator = new ColorValidator();

// 2. 定义自定义颜色
const customColors = {
  primary: '#ff5722',
  secondary: '#00bcd4',
  background: '#263238',
  text: '#eceff1',
};

// 3. 验证颜色
const primaryValidation = colorValidator.validateHex(customColors.primary);
const backgroundValidation = colorValidator.validateHex(customColors.background);

if (!primaryValidation.valid) {
  console.error('主色无效:', primaryValidation.error);
  return;
}

// 4. 验证对比度
const contrastValidation = colorValidator.validateContrast(
  customColors.text,
  customColors.background
);

if (!contrastValidation.valid) {
  console.error('对比度不足:', contrastValidation.error);
  console.error('建议调整背景色或文本色');
  return;
}

// 5. 应用自定义主题
useThemeStore.getState().setTheme('custom');
useThemeStore.getState().updateCustomColors(customColors);

console.log('自定义主题应用成功！');
console.log('对比度等级:', contrastValidation.wcagLevel);

// 6. 清理
colorValidator.destroy();
```

## FontSizeValidator 使用

### 基本验证

```typescript
const validator = new FontSizeValidator();

// 验证像素单位
const result1 = validator.validate('16px');
console.log(result1.valid); // true

// 验证 rem 单位
const result2 = validator.validate('1rem');
console.log(result2.valid); // true

// 验证 em 单位
const result3 = validator.validate('1.5em');
console.log(result3.valid); // true

// 验证百分比单位
const result4 = validator.validate('100%');
console.log(result4.valid); // true
```

### 自定义限制

```typescript
// 创建带自定义限制的验证器
const validator = new FontSizeValidator({
  minFontSize: 12,
  maxFontSize: 64,
  defaultUnit: 'px'
});

// 验证
const result1 = validator.validate('16px');
console.log(result1.valid); // true

const result2 = validator.validate('8px');
console.log(result2.valid); // false
console.log(result2.error); // "Font size 8px is below minimum of 12px"

const result3 = validator.validate('72px');
console.log(result3.valid); // false
console.log(result3.error); // "Font size 72px exceeds maximum of 64px"
```

### 动态调整限制

```typescript
const validator = new FontSizeValidator();

// 设置新的限制
validator.setLimits(14, 48);

// 验证
const result1 = validator.validate('16px');
console.log(result1.valid); // true

const result2 = validator.validate('12px');
console.log(result2.valid); // false
```

### 验证字体大小范围

```typescript
const validator = new FontSizeValidator();

// 验证有效的范围
const result1 = validator.validateRange('12px', '24px');
console.log(result1.valid); // true

// 验证无效的范围（最小值大于最大值）
const result2 = validator.validateRange('24px', '12px');
console.log(result2.valid); // false
console.log(result2.error); // "Minimum size cannot be greater than maximum size"
```

### 字体系统配置

```typescript
import { FontSizeValidator } from './theme/FontSizeValidator';

// 定义字体系统
const fontSizes = {
  xs: '12px',
  sm: '14px',
  base: '16px',
  lg: '18px',
  xl: '24px',
  '2xl': '32px',
  '3xl': '48px',
};

// 创建验证器
const validator = new FontSizeValidator({
  minFontSize: 10,
  maxFontSize: 64,
});

// 验证字体系统
Object.entries(fontSizes).forEach(([name, size]) => {
  const result = validator.validate(size);
  if (result.valid) {
    console.log(`✓ ${name}: ${size}`);
  } else {
    console.error(`✗ ${name}: ${size} - ${result.error}`);
  }
});

validator.destroy();
```

## ZoomController 使用

### 基本缩放

```typescript
const zoom = new ZoomController();

// 设置缩放级别
zoom.setZoomLevel(1.5);
console.log(zoom.getZoomLevel()); // 1.5

// 放大
zoom.zoomIn();
console.log(zoom.getZoomLevel()); // 1.75

// 缩小
zoom.zoomOut();
console.log(zoom.getZoomLevel()); // 1.5

// 重置
zoom.resetZoom();
console.log(zoom.getZoomLevel()); // 1.0

zoom.destroy();
```

### 自定义配置

```typescript
// 创建带自定义配置的缩放控制器
const zoom = new ZoomController({
  initialZoom: 1.0,
  zoomStep: 0.1,      // 更小的步进
  minZoom: 0.5,       // 最小缩放 50%
  maxZoom: 2.0,       // 最大缩放 200%
});

// 使用
zoom.setZoomLevel(0.8);
console.log(zoom.getZoomLevel()); // 0.8

zoom.zoomIn();
console.log(zoom.getZoomLevel()); // 0.9

zoom.destroy();
```

### 键盘快捷键支持

```typescript
import { ZoomController } from './preview/ZoomController';

const zoom = new ZoomController();

// 添加键盘快捷键支持
document.addEventListener('keydown', (event) => {
  if (event.ctrlKey || event.metaKey) {
    switch (event.key) {
      case '+':
      case '=':
        event.preventDefault();
        zoom.zoomIn();
        break;
      case '-':
        event.preventDefault();
        zoom.zoomOut();
        break;
      case '0':
        event.preventDefault();
        zoom.resetZoom();
        break;
    }
  }
});

// 鼠标滚轮缩放
document.addEventListener('wheel', (event) => {
  if (event.ctrlKey || event.metaKey) {
    event.preventDefault();
    if (event.deltaY < 0) {
      zoom.zoomIn();
    } else {
      zoom.zoomOut();
    }
  }
});

zoom.destroy();
```

### 集成到预览面板

```typescript
import { usePreviewStore } from './stores/usePreviewStore';
import { ZoomController } from './preview/ZoomController';

// 初始化缩放控制器
const { initZoomController } = usePreviewStore.getState();
const zoomController = initZoomController();

// 监听缩放变化
usePreviewStore.subscribe((state) => {
  if (state.zoomLevel !== zoomController.getZoomLevel()) {
    // 同步到 UI
    updatePreviewZoom(state.zoomLevel);
  }
});

// UI 控件
function ZoomControls() {
  const zoomLevel = usePreviewStore(state => state.zoomLevel);

  return (
    <div className="zoom-controls">
      <button onClick={() => zoomController.zoomOut()}>-</button>
      <span>{Math.round(zoomLevel * 100)}%</span>
      <button onClick={() => zoomController.zoomIn()}>+</button>
      <button onClick={() => zoomController.resetZoom()}>重置</button>
    </div>
  );
}

// 清理
zoomController.destroy();
```

## SnapshotViewController 使用

### 保存视图状态

```typescript
import { SnapshotViewController } from './snapshot/SnapshotViewController';

const viewState = {
  zoom: 1.5,
  scroll: { x: 100, y: 200 },
  device: 'tablet',
  theme: 'dark',
};

const success = snapshotViewController.saveViewState(viewState);
console.log('视图状态保存:', success);
```

### 恢复视图状态

```typescript
const snapshotId = 'snapshot-abc123';
const viewState = snapshotViewController.restoreViewState(snapshotId);

if (viewState) {
  console.log('恢复视图状态:', viewState);
  // 应用视图状态
  zoomController.setZoomLevel(viewState.zoom);
  scrollTo(viewState.scroll.x, viewState.scroll.y);
  usePreviewStore.getState().deviceSimulator!.setDevice(viewState.device);
}
```

### 同步视图状态

```typescript
const state = {
  zoom: 1.25,
  scroll: { x: 50, y: 150 },
  device: 'tablet',
};

const result = snapshotViewController.syncViewState(state);

if (result.success) {
  console.log('视图同步成功');
  console.log('恢复的缩放:', result.restoredZoom);
  console.log('恢复的滚动:', result.restoredScroll);
} else {
  console.error('视图同步失败:', result.error);
}
```

### 比较视图状态

```typescript
const snapshotId1 = 'snapshot-1';
const snapshotId2 = 'snapshot-2';

const diff = snapshotViewController.compareViewStates(snapshotId1, snapshotId2);

console.log('缩放差异:', diff.zoomDiff);
console.log('滚动差异:', diff.scrollDiff);
console.log('设备改变:', diff.deviceChanged);
console.log('主题改变:', diff.themeChanged);
```

## BoundaryExceptionHandler 使用

### 基本错误处理

```typescript
const handler = new BoundaryExceptionHandler();

try {
  // 可能抛出错误的代码
  riskyOperation();
} catch (error) {
  const result = handler.handle(error as Error, {
    context: 'preview-update',
  });

  if (result.success) {
    console.log('错误已处理');
    if (result.recovered) {
      console.log('已自动恢复');
    }
  }
}

handler.destroy();
```

### 自定义错误处理器

```typescript
const handler = new BoundaryExceptionHandler();

// 注册网络错误处理器
handler.registerHandler('network-error', (error, context) => {
  console.error('网络错误:', error.message);

  // 尝试重试
  if (context.metadata?.retryCount < 3) {
    return {
      success: true,
      recovered: true,
    };
  }

  // 返回降级值
  return {
    success: true,
    recovered: false,
    fallbackValue: getCachedData(),
  };
});

// 注册验证错误处理器
handler.registerHandler('validation-error', (error, context) => {
  console.error('验证错误:', error.message);

  return {
    success: true,
    recovered: true,
    fallbackValue: getDefaultValue(),
  };
});

handler.destroy();
```

### 设置恢复策略

```typescript
const handler = new BoundaryExceptionHandler();

// 为网络错误设置恢复策略
handler.setRecoveryStrategy('network-error', {
  retry: true,
  notify: true,
  log: true,
});

// 为验证错误设置恢复策略
handler.setRecoveryStrategy('validation-error', {
  retry: false,
  fallback: 'default-value',
  notify: false,
  log: true,
});

handler.destroy();
```

### 查看错误历史

```typescript
const handler = new BoundaryExceptionHandler();

// 查看错误历史
const history = handler.getErrorHistory();

history.forEach((record) => {
  console.log(`错误: ${record.error.message}`);
  console.log(`上下文: ${record.context}`);
  console.log(`时间: ${record.timestamp}`);
  console.log(`已恢复: ${record.recovered}`);
});

// 清除历史
handler.clearErrorHistory();

handler.destroy();
```

## 常见问题

### Q: ColorValidator 支持哪些颜色格式？

A: 支持以下格式：
- 十六进制: `#f00`, `#ff0000`
- RGB: `rgb(255, 0, 0)`, `rgba(255, 0, 0, 0.5)`
- HSL: `hsl(120, 100%, 50%)`, `hsla(120, 100%, 50%, 0.5)`

### Q: 如何验证颜色是否符合可访问性标准？

A: 使用 `validateContrast` 方法：
```typescript
const result = colorValidator.validateContrast('#ffffff', '#000000');
console.log(result.wcagLevel); // 'AA' 或 'AAA' 或 'FAIL'
```

### Q: ZoomController 的缩放范围是多少？

A: 默认范围是 0.25 (25%) 到 3.0 (300%)。可以通过 `setMinZoomLevel` 和 `setMaxZoomLevel` 方法自定义。

### Q: 如何在快照中保存和恢复视图状态？

A: 使用 `SnapshotViewController`：
```typescript
// 保存
snapshotViewController.saveViewState({
  zoom: 1.5,
  scroll: { x: 100, y: 200 },
  device: 'tablet'
});

// 恢复
const state = snapshotViewController.restoreViewState(snapshotId);
```

### Q: BoundaryExceptionHandler 如何自动恢复错误？

A: 在构造函数中启用 `autoRecovery`：
```typescript
const handler = new BoundaryExceptionHandler({
  autoRecovery: true,
  retryAttempts: 3
});
```

### Q: 如何清理验证器的资源？

A: 调用 `destroy()` 方法：
```typescript
const validator = new ColorValidator();
// 使用验证器...
validator.destroy();
```

### Q: 集成测试失败怎么办？

A: 1. 检查依赖是否正确安装
2. 确保测试环境配置正确
3. 查看测试日志获取详细错误信息
4. 运行单个测试文件进行调试
