## 📋 测试执行

### 运行 E2E 测试

```bash
# 运行所有 E2E 测试
npm run test:e2e

# 运行特定测试文件
npx playwright test e2e/complete-user-flow.spec.ts
npx playwright test e2e/performance-benchmark.spec.ts
npx playwright test e2e/security-tests.spec.ts

# UI 模式运行
npm run test:e2e:ui

# 有头模式运行
npm run test:e2e:headed
```

### 性能测试执行

```bash
# 运行性能基准测试
npx playwright test e2e/performance-benchmark.spec.ts

# 生成性能报告
npx playwright test --reporter=html
```

### 安全测试执行

```bash
# 运行安全测试
npx playwright test e2e/security-tests.spec.ts

# 生成安全报告
npx playwright test e2e/security-tests.spec.ts --reporter=json
```
